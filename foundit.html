Foundit


Role: Full Stack Developer


Skills: Ruby, Javascript, Rails, PostgreSQL, jQuery/AJAX, Redis, Paperclip

Foundit is a lost and found website created at the Civic Hackathon. It allows the SF community to enter in lost and found items so that people can search through using location, as well as, a strong description algorithm. We built the site using Rails, Ruby and Javascript.
