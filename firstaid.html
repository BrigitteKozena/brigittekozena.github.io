Firs+Aid


Role: Full Stack Developer


Skills: Ruby, Javascript, Rails, PostgreSQL, jQuery/AJAX, Redis, Stripe, Devise, Twillio, D3, Google Maps API

First Aid notifies users when a disaster occurs and facilitates the ability to send donations to nearby local charities. When starting this project we were aware that users are constantly being overflooded with data. We wanted to ensure that these disasters were not overlooked and thus, presented the data visually using D3. We used websockets to plot the disasters in realtime and the google maps API in order to locate nearby charities. We used Twillio and Redis to notify the user by text and email when a disaster hits. We also implemented the ability to sign up through facebook and share the donations you have made. The overall platform was built using Rails, Ruby and Javascript.
