E-Commerce


Role: Full Stack Developer


Skills: Ruby, Javascript, Rails, PostgreSQL, jQuery/AJAX, Redis, Stripe, Devise, Paperclip

This particular project was based on the Amazon shopping site. For this particular assignment we used Ruby, Rails, Java Script, jQuery/AJAX, HAML & CSS. We additionally, implemented Redis for emails and Stripe for payments.

Our client requested that we build a shopping site similar to Amazon. This included an inventory management section for admin use only.

Users had to be able to browse through products in a grid like fashion and add to cart when they desired. User authentication was required, as well as emails sent when registering and checking out. We also were requested to disable the ability to add products when stock quantity was exceeded.

An admin had to be able to add products to multiple categories, as well as edit and delete them.

The site had to also be mobile friendly.
